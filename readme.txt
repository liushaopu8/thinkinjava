    1. 寄存器是最快的存储区位于处理器内部，数量及其有限所以按需分配
        Java哪些变量会被存储在寄存器中呢？

    2. 堆栈指针向下移动分配洗的内存，向上移动释放内存。创建程序时必须知道存储在堆栈内所有项的确切生命周期，以便上下移动堆栈指针
        堆栈用于存储对象引用，堆用于存放所有的Java对象。
        堆不同于堆栈好处是编译器不需要知道存储的数据在堆里存活多长时间。
        堆进行存储分配和清理可能比用堆栈进行存储分配需要更长的时间

    3. 常量值通常直接存放在程序代码内部，这样做是安全的，因为它们永远不会被改变。
        程序代码内部就是ROM(只读存储器)吗，和RAM有什么区别呢？

    4. Java对“基本”类型采取和C/C++相同的方法，不用new创建而是创建一个并非是引用过的“自动”变量，这个变量直接存储“值”，并置于堆栈中。
        常量和基本类型的区别是什么呢？  String定义的变量和常量，基本类型，new对象的区别是什么呢？

    5. 所有数值类型都有正负号，所以不要去找无符号的数值类型。
        为什么char 是从0 ~  2^16-1

    6. 在C/C++中使用数组是很危险的，因为C和C++中的数组就是内存块，若果访问内存块之外的数组或在初始化前使用内存都会产生难以预料的后果。
       Java确保数组会被初始化，而且不能在它的范围之外被访问。 Java的主要目标之一是安全性，还有哪些安全性呢？

    7. 若类的某个成员是基本数据类型，即使没有进行初始化，Java也会确保它获得一个默认值，如果是局部变量得到的可能是任意值而不会被自动初始化为0
        许多C++编译器会对未初始化变量给与警告，而Java则视为错误
